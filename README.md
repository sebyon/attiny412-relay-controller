# ATtiny412 Relay Control

This project is a small bare-metal firmware example for the ATtiny412 using avr-gcc and PlatformIO.

Primary use case is for incorporating smart switching using a relay for true bypass for guitar pedal effects.

> [!NOTE]
> Currently a work in progress. Please use examples in references for examples that will actually work.
